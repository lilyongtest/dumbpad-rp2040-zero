# dumbpad-rp2040-zero

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

This project is a modified version of the **[dumbpad v3x](https://github.com/imchipwood/dumbpad)**, customized to support the **Waveshare RP2040-Zero** instead of the traditional Pro Micro.

## üìù Key Changes
- **Controller:** Swapped Pro Micro for RP2040-Zero (more memory, faster performance).
- **USB:** Benefit from the RP2040-Zero's USB-C connection.
- **Compatibility:** Maintains the 3x4 + Encoder layout and RGB underglow.
